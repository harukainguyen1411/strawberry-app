Strawberry — Offline Secret Encryptor

Runs entirely in this browser. No network calls. Output is age-encrypted ASCII-armored ciphertext that only the Windows agent box can decrypt. Paste the ciphertext into your agent chat and the agent will pipe it through tools/decrypt.sh.

Recipient (Windows agent runtime)

Baked in at build time from secrets/recipients.txt. If you regenerate the Windows keypair, update both this file and the recipients file together. Always verify tools/encrypt.html.sha256 before opening.

Plaintext
Ciphertext (paste this into the agent chat)

Library: age-encryption v0.2.4 (vendored, no CDN). Bundled to tools/age-bundle.js. Verify integrity with the SHA256 sidecar files before each use.
